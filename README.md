# film-site-metadata-archive
本仓库用于整理影视类网站的元数据信息，记录网络中公开可访问的电影、电视剧相关站点的基础资料。整理内容包括站点名称、访问链接、内容方向与页面结构特征，便于进行信息留存与后续分析。项目不涉及任何影视资源文件的存储、播放或传播，仅用于站点层面的信息归档与技术研究。所有资料均来源于公开网络页面，不构成任何形式的推荐、评价或商业引导。
